# Bezier Surface Renderer

A C# Windows Forms application for rendering and interacting with 3D Bezier surfaces.

## Features

- **3D Rendering:** Renders 4x4 cubic Bezier surfaces using a custom triangle mesh filler.
- **Rendering Modes:** Choose between Polygon, Mesh grid, and Filled rendering.
- **Lighting Model:**
  - Configurable material properties (Kd, Ks, Shininess).
  - Multiple light source types: Directional, Point, and Spotlight.
  - Interactive light position and color.
  - Light source animation.
- **Texturing & Materials:**
  - Solid color or Texture mapping support.
  - Normal mapping for detailed surface relief.
- **Interactive Controls:**
  - Adjust triangulation density.
  - Rotate the view (Alpha, Beta angles).
  - Load custom control points, textures, and normal maps.
  - Animate surface control points.
